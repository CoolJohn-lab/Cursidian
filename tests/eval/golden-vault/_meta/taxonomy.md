# Taxonomy

## Categories

- projects - platform-level initiatives, one hub page per platform.
- concepts - reusable ideas and patterns shared across the platform.
- entities - concrete systems, source feeds, or tables.
- skills - how-to procedures for a named task.

## Core tags

- ingestion, egress, contract, medallion, worker-data-mart, bighand, forecast, orchestration, schema-drift
